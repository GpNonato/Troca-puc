### 4.2. Tecnologias

Para implementar a solução proposta, utilizaremos uma série de tecnologias que abrangem desde o desenvolvimento do front-end até a gestão do banco de dados e o deploy da aplicação. Abaixo, estão listadas as principais tecnologias envolvidas no projeto:

- **SGBD (Sistema de Gerenciamento de Banco de Dados)**:
  - **PostgreSQL**: Utilizado para armazenar e gerenciar os dados do projeto. O PostgreSQL será a base de dados relacional, onde as informações estruturadas serão mantidas.

- **Front-end**:
  - **HTML, CSS e JavaScript**: Essas tecnologias compõem a base do front-end, responsáveis por estruturar, estilizar e adicionar interatividade à interface do usuário. 
  
- **Back-end**:
  - **Java com SpringBoot**: Será utilizado para construir a lógica de negócio da aplicação, gerenciar rotas e requisições, garantir a comunicação com o banco de dados, além de hospedar conteúdos estáticos.

- **Deploy**:
  - **Coolify**: A aplicação será publicada e mantida online usando o servidor do participante Pedro Kalil, que é gerenciada usando o coolify.
  - **Github Actions**: O github actions será utilizado para fazer o deploy de novas releases para o servidor.

Além disso, outras ferramentas e serviços auxiliarão no desenvolvimento, como:

- **Frameworks e Bibliotecas**: 
  - **SpringBoot** (para o back-end em Java)
  - **Bootstrap** (para estilização adicional no front-end)

- **Ferramentas de desenvolvimento**:
  - **IDE**: Utilizaremos o **Visual Studo Code** e o **IntelliJ IDEA** tano para o desenvolvimento do back-end em Java e o quanto para trabalhar com HTML, CSS e JavaScript no front-end. A escolha fica a cabo de cada desenvolvedor envolvido no projeto.
  - **Git** e **GitHub**: Para controle de versão e colaboração.
  - **GitHub Projects**: Para controle de fluxo e execução das sprints e suas issues.

Essas tecnologias foram escolhidas para garantir uma aplicação robusta, escalável e fácil de manter.
De acordo com orientação dos orientadores do projeto.

| Front end      | HTML+CSS+JS (Bootstrap) |     |
| Back end       | Java SpringBoot |
| Deploy         | Coolify+Github Actions    |
